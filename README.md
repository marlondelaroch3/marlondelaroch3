Soy **Desarrollador Frontend y Fundador** con casi 4 años de experiencia creando aplicaciones web de alto impacto. Actualmente, estoy dedicado al 100% a la construcción de **RochCode**, mi propia factoría de software, y al desarrollo de **Comuno**, un sistema especializado para propiedad horizontal en Colombia.

Me apasiona la **optimización de procesos** y la **automatización**. Mi objetivo es encontrar siempre la forma más eficiente y elegante de resolver problemas complejos a través del código.

### 🚀 En qué estoy trabajando hoy
- 🛠️ Construyendo **Comuno**: Transformando la comunicación entre residentes y administraciones.
- 🏗️ Escalando **RochCode**: Desarrollando soluciones de software a medida.
- 🤖 Automatizando todo lo posible con **n8n**.
